실습과제1

STL 라이브러리에서 컨테이너 클래스 , 이터레이터 클래스 , 알고리즘 함수의 역할을 각각 설명하시오 
컨테이너 클래스 - 데이터를 저장하는 자료 구조를 구현한 템플릿 클래스 
이터레이터 클래스 - 컨테이너의원소에 대한 포인터를 저장하는 클래스 
알고리즘 - 컨테이너 원소에 대한 복사, 검색, 삭제, 정렬 등을 구현한 템플릿 함수

포인터와 iterator 의 공통점과 차이점은 각각 무엇인가 
공통점 - 증감 연산(++, --)을 통해 다음/이전 요소로 이동 가능 
차이점 - 포인터는 메모리 주소를 저장하는 변수 이지만 이터레이터는 컨테이너의 요소를 가리키기 위한 추상화된 객체이다

vector 클래스의 erase, insert 멤버함수의 매개변수 리턴값 , 기능을 설명하시오 
erase - 지정한 위치의 원소를 삭제후 뒤의 요소들을 앞으로 이동시켜 재배치 시킨다 
insert - 지정한 위치에 원소를 삽입한다
